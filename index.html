pyflakes.vim: check Python code on the fly

real-time highlighting for name errors, missing imports, and more

pyflakes.vim is a vim plugin using pyflakes to highlight real-time mistakes in your Python code.

:PyflakesUpdate runs PyFlakes on your code and updates the QuickFix list. But,

  • dd, dw, and leaving insert mode are bound to update PyFlakes as well so you don't have to think about it.
  • use your favorite quickfix commands like :cc to jump between errors

Do :help pyflakes-vim for more info, or take a look at the README or the vimscript page.
